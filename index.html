📲 Tải & cài đặt cấu hình iOS

Nhấn nút bên dưới để tải file cấu hình .mobileconfig.

Tải / Cài đặt mobileconfig

Nếu bấm mà nó chỉ hiện chữ (không tải), hãy bấm nút Share của Safari → Save to Files hoặc mở link bằng Safari (không dùng in-app browser).

Cách cài trên iPhone/iPad:

  1. Mở trang này bằng Safari.
  2. Bấm nút tải ở trên.
  3. Vào Cài đặt (Settings) → Profile Downloaded (hoặc General → VPN & Device Management).
  4. Chọn profile vừa tải → Install.
